## Overview

NullSec Windows is an exclusive security toolkit built specifically for Microsoft Windows systems using native languages and APIs.

## Languages Used

| Language | Purpose | Why |
|----------|---------|-----|
| **C#** | Main toolkit | Native .NET with full Windows API access |
| **F#** | Functional analysis | Type-safe security analysis tools |
| **PowerShell** | Automation | Deep Windows integration |
| **Batch** | Legacy support | Quick scripts for older systems |
